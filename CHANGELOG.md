# Changelog

## 2026-04-01

### 启动与构建

- Gradle 固定优先使用本地 JDK 21
- 关闭 Java toolchain 自动下载，避免命令行构建反复拉取远程 JDK
- 开启 Gradle build cache 和 configuration cache
- 延后标签颜色修正逻辑，减少首页启动阶段数据库负担

### 默认入口

- App 默认启动进入日程页
- 日程页默认打开周视图

### 日程页体验

- 日 / 周 / 月视图左右切换增加主体内容区动画
- 第一行视图切换按钮、第二行日期信息和底部导航栏不参与动画
- 日视图顶部日期栏固定，不随内容滚动
- 月视图在当天详情模式下，左右切换只动画下半部分详情区域

### 周视图

- 每小时时间轴高度缩短到原来的约 4/5
- 卡片结束时间改为左下角显示
- 修复底部时间轴 `22:00` 显示被遮挡的问题
- 首屏先渲染稳定骨架，再异步填充课程与任务
- 课程节次数据未到前不先渲染课程块，避免二次抖动
- 时间轴范围优先由课表时段决定，避免课程整体重排
- 骨架层与覆盖层分离，只在布局变化时重建主体
- 周数据无变化时不重复整块重绘
